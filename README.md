## 🚀 Implementation Steps

1. Explored Bootstrap 5 documentation and examples
2. Selected components like navbar, cards, carousel, and forms
3. Combined components creatively instead of copying templates
4. Designed 3 responsive pages
5. Added custom CSS for styling and hover effects
6. Tested responsiveness on multiple screen sizes
7. Uploaded project to GitHub
8. Deployed using GitHub Pages / Netlify
